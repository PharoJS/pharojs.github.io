How to install PharoJS?

Install can be done inside a Pharo image by evaluating the following in a playground:

Gofer it
smalltalkhubUser: 'noury' project: 'PharoJS';
addPackage: #ConfigurationOfPharoJS ;
loadStable.

This will install the most recent stable version. To install the latest development version, please use loadBleedingEdge instead of loadStable.

Where can I find the PharoJS documentation?

The PharoJS documentation is available in the dedicated page. It includes conference papers and slides of talks presenting different facets of PharoJS.
